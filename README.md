
# Mission Control AI 

## Sobre o Projeto

O Mission Control AI é um sistema inteligente de monitoramento espacial desenvolvido em Python para simular o controle operacional de uma missão espacial experimental sustentável.

O sistema analisa dados simulados em tempo real, identifica situações críticas automaticamente e executa respostas automatizadas para auxiliar na segurança da missão.

O projeto aplica conceitos de:

* Programação em Python
* Algoritmos
* Pensamento computacional
* Inteligência artificial introdutória
* Automação
* Sustentabilidade energética
* Monitoramento operacional

---

# Objetivo

Desenvolver uma plataforma funcional capaz de:

* Monitorar sensores da missão espacial
* Simular dados operacionais
* Gerar alertas automáticos
* Executar análises inteligentes
* Armazenar histórico operacional
* Auxiliar na tomada de decisão

---

# Funcionalidades

✅ Cadastro manual de sensores
✅ Simulação automática de sensores
✅ Monitoramento de temperatura

✅ Monitoramento estrutural
✅ Geração automática de alertas
✅ Inteligência artificial introdutória

# Tecnologias Utilizadas

* Python 3
* Biblioteca random
* Biblioteca time

# Inteligência Artificial do Sistema

O Mission Control AI utiliza estruturas lógicas para:

* Detectar situações críticas
* Gerar alertas automáticos
* Executar respostas automatizadas
* Classificar o status geral da missão

Exemplos:

* Ativação automática do sistema de resfriamento
* Economia automática de energia
* Reconexão automática da comunicação
* Liberação emergencial de oxigênio


# Sustentabilidade

O projeto possui foco em sustentabilidade e eficiência energética.

A missão espacial utiliza energia solar como principal fonte energética, simulando aplicações modernas de energias renováveis na indústria espacial.


# Conclusão

O Mission Control AI demonstra como a programação pode ser utilizada para desenvolver soluções inteligentes aplicadas ao monitoramento espacial moderno.

O projeto une automação, análise operacional, inteligência computacional e sustentabilidade em uma plataforma funcional e organizada.
